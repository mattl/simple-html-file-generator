# Generate a simple HTML document right from your browser! 

* No command-line or terminal knowledge or experience needed
* No database
* Basic HTML to get you started

A good place to host a website like this is <a href="https://neocities.org">Neocities</a>.

## Getting started

Everything is generated on your computer, so no need to download/install anything.

* Visit https://mattl.github.io/simple-html-file-generator/ or download/clone this repository

[Please send any bugs or improvements via GitHub issues.](https://github.com/mattl/simple-html-file-generator/issues).
